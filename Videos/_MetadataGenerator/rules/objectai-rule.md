# ObjectAI — Metadata Rule

## Channel Identity
- **Name:** objectwithbrainsai
- **Instagram:** @objectwithbrainsai
- **Facebook:** objectwithbrainsai
- **YouTube:** NOT USED
- **Niche:** Everyday objects with AI personalities — talking fruits, furniture, gadgets
- **Target audience:** 18-35, viral content lovers, tech-curious, humor-seekers

## Content Type
- **Format:** Instagram Reels + Facebook Reels + Carousels
- **Style:** Photorealistic objects with personified voices (Veo 3.1 generated)
- **Language:** English primary, some Hindi/Hinglish

## Tone
- Witty, sarcastic, unexpected
- Absurdist humor
- Viral-first
- Character-driven (each object has distinct voice)

## Instagram Caption Template
```

[Short dialogue excerpt from video]

```

## Instagram Hashtags (3-5)
- Niche: #objectwithbrainsai
- Medium: #aigenerated #viralreels
- Broad: #funnyvideos #reels

## Facebook Post Template
```
😂 [Hook line — what the object said]

[Quote from video]

```

## Hook Writing Rules
- First 2 sec: Object revealed with surprising voice/expression
- Text overlay with the zinger line
- Character's personality must land immediately

## Viral Triggers
- Relatability ("we've all thought this")
- Absurdity (objects saying human things)
- Tag-a-friend moments
- Save-worthy one-liners

## SEO Keywords (always include 2-3)
- ai generated content, object with brains ai, viral reels, ai funny, ai characters, ai personification, object talks

## Caption Style Rules
- Always include dialogue quote from video
- Always include tag-a-friend CTA
- Keep it short (under 150 chars ideal)
- Emojis sparingly (2 max)
- No hashtags in Facebook post body — only at end

## What NOT to do
- No long explanations (breaks the joke)
- No disclaimers about AI (fans know)
- No Gappu-style emotional tone (too sweet for this channel)
- No YouTube metadata (channel isn't on YouTube)
