# YouTube SEO Rules (extracted from claude-youtube skill)

## Title Rules
- Length: 60-100 characters (ideal: 70 chars)
- Primary keyword in **first 40 characters**
- 3 title variants required:
  - **A. Search-optimized**: keyword-forward, clear intent
  - **B. Browse/curiosity**: emotional trigger, clickbait hook
  - **C. Hybrid (recommended)**: keyword + compelling frame
- No ALL CAPS (feels spammy)
- Power words: Secret, Hidden, Truth, Why, How, Never, Shocking
- Hindi channels: use both Hindi + Roman script mix for reach

## Description Rules
- Max 5000 chars, aim 800-1500 chars
- **First 2 lines** visible before "Show More" — make them hook
- Primary keyword in **first 25 words**
- Structure:
  - Line 1-2: Hook (visible)
  - Para 2: Expanded context with secondary keywords
  - TIMESTAMPS section (chapters)
  - LINKS section
  - Channel boilerplate
  - Hashtags at BOTTOM (3-5)

## Tags Rules
- 10-15 tags total
- Max 500 chars combined
- Order: exact keyword → variations → broader topics
- Include misspellings only if common
- Include channel brand name as tag

## Chapters Rules
- Start at 0:00 always
- Minimum 3 chapters
- Keyword-rich chapter labels (not generic)
- For long videos (10+ min): every 1-2 min a chapter

## Hashtags Rules (YouTube-specific)
- 3-5 hashtags in description (NOT title)
- First = most specific, last = broadest
- YouTube shows top 3 above title

## Hook Rules (First 3-5 sec)
- State value proposition immediately
- Create curiosity gap
- Show don't tell
- For Shorts: visual hook in frame 1

## Shorts-specific
- Vertical 9:16
- Under 60 sec
- Hook in first 2 sec (not 5)
- Loop-friendly ending (encourages replay)
- Captions burned in (sound-off viewing)
